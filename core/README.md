# Серверная часть

1) Для работы бекенда требуется развернутый локально PostgreSQL сервер на порту 5432 с кредами, указанными в .env.\
Или, можно самостоятельно развернуть локальную БД PostgreSQL с уже созданной БД/пользователями, введя команду ниже
```
docker compose up -d
```

2) Для обеспечения возможности запуска бекенда, необходимо установить зависимости, перечисленные в requirements.txt\
Для установки через pip введите команду ниже
```
pip install -r requirements.txt
```

3) Для запуска бекенда локально необходимо ввести команду ниже:
```
uvicorn app.main:app --reload
```
